# Sistema de Tipografia

Os componentes de Tipografia fornecem estilos de texto consistentes em todo o DSC. Cada componente aplica estilos predefinidos baseados nos tokens do design system, garantindo hierarquia visual adequada e experiência de leitura otimizada.

## Propriedades Aplicadas

Cada componente de tipografia aplica automaticamente as seguintes propriedades CSS:

- **fontFamily**: Define a família tipográfica apropriada
- **fontWeight**: Aplica o peso da fonte (regular, semibold, bold)
- **fontSize**: Define o tamanho da fonte específico para cada categoria
- **lineHeight**: Estabelece a altura da linha para legibilidade otimizada
- **letterSpacing**: Ajusta o espaçamento entre caracteres quando necessário

Essas propriedades são derivadas dos tokens do design system e garantem consistência visual em toda a aplicação.

## Categorias de Tipografia

O sistema de tipografia é organizado em categorias específicas, cada uma otimizada para diferentes contextos de uso:

### Display
Texto grande para títulos principais e displays proeminentes. Usado em páginas de destaque, headers principais e elementos que precisam chamar máxima atenção.

<Canvas of={TypographyStories.DisplayComponents} />

```tsx
<DisplayStandard>Display Standard</DisplayStandard>
<DisplayStandardSemibold>Display Standard Semibold</DisplayStandardSemibold>
<DisplaySmall>Display Small</DisplaySmall>
<DisplaySmallSemibold>Display Small Semibold</DisplaySmallSemibold>
```

### Display Numerical
Texto grande otimizado para exibição de valores numéricos como valores monetários, percentuais e métricas importantes. Possui espaçamento e kerning específicos para melhor legibilidade de números.

<Canvas of={TypographyStories.DisplayNumericalComponents} />

```tsx

<DisplayNumericalStandardSemibold>123.456,78</DisplayNumericalStandardSemibold>
<DisplayNumericalSmall>123.456,78</DisplayNumericalSmall>
<DisplayNumericalSmallSemibold>123.456,78</DisplayNumericalSmallSemibold>
```

### Title
Títulos de seção e subtítulos para organização hierárquica do conteúdo. Ideal para estruturar páginas e dividir seções de informação.

## Como usar

Cada componente aceita todas as propriedades padrão do `Text` do Tamagui para estilização e comportamento adicionais:

```tsx
import { BodyStandard, TitleLarge } from '@superapp-caixa/dsc-library';

// Uso básico
<TitleLarge>Meu Título</TitleLarge>
<BodyStandard>Texto do conteúdo principal</BodyStandard>

// Com propriedades adicionais do Tamagui
<BodyStandard color="$accent9" textAlign="center">
  Texto centralizado com cor personalizada
</BodyStandard>

// Aplicando temas
<TitleLarge theme="success">Título com tema de sucesso</TitleLarge>
```
