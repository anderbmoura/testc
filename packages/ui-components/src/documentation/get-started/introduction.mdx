# {LIBRARY_NAME}

Esta biblioteca foi desenvolvida tendo como base o Core do framework <a href="https://tamagui.dev/" target="_blank" rel="noopener noreferrer">Tamagui</a>, que é utilizado para a construção da arquitetura, componentes, layouts e templates de tela dos aplicativos móveis da CAIXA.

## Características Principais

- **Consistência de Design**: Construído seguindo os padrões de design da CAIXA.
- **Integração com Tamagui**: Aproveita o sistema de estilização e performance do Tamagui.
- **Suporte a Temas**: Variantes de tema claro e escuro, dentre outros.

## Uso com Tamagui

> **Recomendação**: Utilize **sempre que possível** os componentes de construção e estilo de layout do Tamagui (`View`, `Text`, `Stack`, etc.) em detrimento dos componentes React Native.

### Por que usar componentes Tamagui?

- **Performance otimizada**: Sistema de styling compilado em tempo de build.
- **Consistência de tema**: Compartilhamento do mesmo sistema de tokens.
- **Bundle size reduzido**: Tree-shaking eficiente dos estilos não utilizados.

### Exemplo recomendado:

```tsx
import { View, XStack, YStack } from '@tamagui/core';
import { Button, Input, TitleLarge } from '@superapp-caixa/dsc-library';

export function MinhaTela() {
  return (
    <YStack padding="$medium" space="$small">
      <TitleLarge>
        Título da Tela
      </TitleLarge>

      <XStack space="$tiny" alignItems="center">
        <Input flex={1} placeholder="Digite aqui..." />
        <Button>Buscar</Button>
      </XStack>
    </YStack>
  );
}
```
